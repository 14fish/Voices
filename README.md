# Voices

The Voices is an open-source app that lets you collect labeled data for training. 

User uploads a txt file and Voices finds five sentences from txt file and divides into 5 cards.

In each card user records ownself voice, when up to 5 cards done (cards are skippable), user can download every single voice named as text inside card.
